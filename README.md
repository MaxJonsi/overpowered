# Overpowered

Мод для Minecraft 1.21.1 (NeoForge): суперспособности, анимированное оружие, мобы, магия и постройки. Кастомные 3D-модели и анимации на [GeckoLib](https://github.com/bernie-g/geckolib).

## Контент

### Ямато (катана)
- ЛКМ: комбо из трёх ударов с широким уроном по площади
- ПКМ: Judgement Cut — сфера пространственных разрезов в точке взгляда (до 24 блоков)
- Shift + ПКМ (держать 1.5 сек): Judgement Cut End — катана убирается в ножны под музыку, затем вся область в радиусе 24 блоков рассекается бесконечными разрезами
- R: рывок с клинком, режет всех на пути

### Войд
- V: превращение — персонаж становится полностью чёрным
- F: смотря на любого моба или игрока — цель умирает, на земле навсегда остаётся её тень

### Опустошитель (ракетница)
- R: переключение режимов
- Режим 1: самонаводящиеся ракеты. H — пометить цель; без метки ракета сама ищет ближайшую
- Режим 2: ядерный авиаудар — выбери точку взглядом (до 300 блоков), заряд падает с неба и выжигает сферу диаметром 100 блоков
- Режим 3: лазер аннигиляции — удерживай ПКМ, луч разрушает любой блок за 0.1 сек

### Шесть глаз (техники Годжо)
- R: переключение техник
- Синий: вихрь притяжения в точке взгляда
- Красный: волна отталкивания вокруг себя
- Пустотный пурпур: летящая сфера, стирающая блоки и врагов на пути
- Расширение территории: сфера 50 блоков с текстурой космоса; никто внутри не может двигаться, а хозяин убивает любого с одного удара (15 секунд, кулдаун 90 сек)

## Музыка Bury the Light

Из-за авторских прав трек не входит в репозиторий: вместо него играет синтезированный райзер. Чтобы звучал оригинал, положи свой ogg-файл (моно или стерео) в `src/main/resources/assets/overpowered/sounds/weapon/bury_the_light.ogg` в локальной сборке или сделай ресурспак с файлом `assets/overpowered/sounds/weapon/bury_the_light.ogg` — начни файл с момента "I am the storm that is approaching".

## Скачать

Готовый jar собирается автоматически:

- Стабильные версии: [Releases](https://github.com/MaxJonsi/overpowered/releases)
- Свежие сборки с каждого коммита: [Actions](https://github.com/MaxJonsi/overpowered/actions) -> выбрать последний запуск -> Artifacts

## Установка

1. Установить [NeoForge](https://neoforged.net/) для Minecraft 1.21.1
2. Скачать [GeckoLib](https://modrinth.com/mod/geckolib/versions?g=1.21.1&l=neoforge) для NeoForge 1.21.1
3. Положить оба jar-файла в папку `mods`

## Стек

- Minecraft 1.21.1, NeoForge 21.1.235
- GeckoLib 4.9.2 (анимации, 3D-модели)
- Java 21, Gradle (ModDevGradle)
- Модели и анимации в формате Blockbench (geo.json / animation.json)

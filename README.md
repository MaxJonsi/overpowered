# Overpowered

Мод для Minecraft 1.21.1 (Fabric): сверхспособности, анимированное оружие и разрушительная магия. Кастомные 3D-модели и анимации работают на [GeckoLib](https://github.com/bernie-g/geckolib), а позы игрока - на [Player Animator](https://modrinth.com/mod/playeranimator).

## Контент

### Ямато (катана)
- ЛКМ: комбо из трёх ударов с широким уроном по площади
- ПКМ: Judgement Cut — сфера пространственных разрезов в точке взгляда (до 24 блоков)
- Shift + ПКМ (держать 1.5 сек): Judgement Cut End — катана убирается в ножны под музыку, затем вся область в радиусе 24 блоков рассекается бесконечными разрезами
- R: рывок с клинком, режет всех на пути
- При удержании катаны и во время приёмов синхронизируются двуручные позы игрока; заряд Judgement Cut End показан отдельной HUD-шкалой

### Войд
- ПКМ со Сферой Пустоты: включить или выключить форму Войда
- ЛКМ в форме Войда: стереть живую цель в прицеле на расстоянии до 32 блоков; на земле остаётся её тень
- Валидная цель отмечается красным прицелом; форма даёт полёт и защищает от урона при падении

### Опустошитель (ракетница)
- R: переключение режимов
- Режим 1: самонаводящиеся ракеты. H — пометить цель; без метки ракета сама ищет ближайшую
- Режим 2: ядерный авиаудар — выбери точку взглядом (до 300 блоков), заряд падает с неба и выжигает сферу диаметром 200 блоков, постепенно загружая нужные чанки с временными билетами и серверным бюджетом до 20 000 позиций за тик
- Режим 3: лазер аннигиляции — удерживай ПКМ; непрерывный луч идёт до края загруженной области и разрушает блок за 1 тик

### Шесть глаз (техники Годжо)
- Для применения и переключения техник должна быть надета Маска Годжо
- R: переключение техник
- Синий: после короткого заряда между ладонями создаёт вихрь притяжения в точке взгляда
- Красный: волна отталкивания вокруг себя
- Пустотный пурпур: летящая сфера, стирающая блоки и врагов на пути
- Расширение территории: космическая сфера с чёрной дырой в центре; никто внутри не может двигаться, а хозяин убивает любого с одного удара (15 секунд, кулдаун 90 сек)

## Музыка Bury the Light

В текущей ветке находится предоставленный для проекта 12.8-секундный фрагмент `assets/overpowered/sounds/weapon/bury_the_light.ogg`. Перед публичным распространением сборки необходимо самостоятельно проверить права на этот файл. Его можно заменить собственным лицензированным звуком через ресурспак по тому же пути.

## Скачать

Готовый jar собирается автоматически:

- Стабильные версии: [Releases](https://github.com/MaxJonsi/overpowered/releases)
- Свежие сборки с каждого коммита: [Actions](https://github.com/MaxJonsi/overpowered/actions) -> выбрать последний запуск -> Artifacts

## Установка

1. Установить [Fabric Loader](https://fabricmc.net/use/installer/) для Minecraft 1.21.1
2. Скачать [Fabric API](https://modrinth.com/mod/fabric-api/versions?g=1.21.1) для 1.21.1
3. Скачать [GeckoLib](https://modrinth.com/mod/geckolib/versions?g=1.21.1&l=fabric) для Fabric 1.21.1
4. Положить три jar-файла (`fabric-api`, `geckolib`, `overpowered`) в папку `mods`; Player Animator уже упакован внутрь мода

## Стек

- Minecraft 1.21.1, Fabric Loader + Fabric API
- GeckoLib 4.9.2 (анимации, 3D-модели)
- Player Animator 2.0.4 (двуручные позы и анимации игрока, включён в jar)
- Java 21, Gradle (Fabric Loom, маппинги Mojang)
- Модели и анимации в формате Blockbench (geo.json / animation.json)
